## Features

- **Auto Capture** - Takes a photo when Mac wakes up/unlocks
- **Manual Capture** - Trigger capture anytime
- **Custom Save Location** - Choose your photo storage directory
- **One Photo Per Day** - Avoids duplicates
- **Countdown Timer** - 4-second countdown with sound
- **Menu Bar App** - Runs in menu bar, no Dock icon
- **Launch at Login** - Auto start when Mac boots

## Requirements

- macOS 13.0+
- FaceTime HD camera

Swift 5.9+ · SwiftUI + AppKit · AVFoundation
