## 功能特性

- **自动拍照** - Mac 唤醒/解锁时自动拍摄
- **手动拍照** - 支持随时手动触发
- **自定义保存路径** - 可自由选择照片保存目录
- **每日仅拍一张** - 避免重复
- **倒计时提醒** - 4 秒倒计时，带音效
- **菜单栏应用** - 运行于菜单栏，不占用 Dock
- **开机自启动** - 支持开机自动启动

## 系统要求

- macOS 13.0+
- FaceTime 高清摄像头

## 技术栈
